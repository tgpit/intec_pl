Этот JS-скрипт необходим для удобной выборки данных по новым полям.
Нужно скопировать весь код в консоль (на странице https://autoload.avito.ru/format/realty/ или других категорий),
предварительно указав нужные поля в переменной fields (строка56),
и потом там же в консоли забрать результат, которые вставить в три разных места (для каждого плагина):
1) в метод getFields
2) в метод processElement
3) в lang-файл (при необходимости - подкорректировать названия!)
